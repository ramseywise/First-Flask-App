# Terminal Commands

## UNIX commands
##### make new directory
mkdir *name* 
##### change directory
cd *name* 
##### list current directory
pwd 
##### list files in directory
ls
##### move file
mv *name* 
##### copy file
cp *name* 
##### lets you change read, write and execute permissions of file
chmod *name* 
##### compresses file
gzip *name* 
##### uncompress files
gunzip *name*
